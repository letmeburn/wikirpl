Roadmap Belajar

Pilih level yang sesuai dengan kemampuanmu dan mulai perjalanan belajar

Level 1

🌱 Pemula

  • Dasar Pemrograman
  • Algoritma & Flowchart
  • HTML, CSS, JavaScript
  • Git & GitHub Dasar
  • Database SQL Dasar

⏱️ Estimasi: 3-6 bulan

Level 2

🚀 Menengah

  • OOP & Design Patterns

  • REST API & Testing

⏱️ Estimasi: 6-12 bulan

  • DevOps & CI/CD
  • Cloud Computing
  • System Design
  • Performance Optimization

⏱️ Estimasi: 12+ bulan
