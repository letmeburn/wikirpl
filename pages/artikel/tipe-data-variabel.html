🎯 Apa itu Variabel?

Variabel adalah container untuk menyimpan data dalam program. Bayangkan variabel seperti kotak yang diberi label, dimana kita bisa menyimpan dan mengambil nilai.

Deklarasi Variabel

// JavaScript
let name = "John";
const age = 25;
var city = "Jakarta";

// Python
name = "John"
age = 25
city = "Jakarta"

// Java
String name = "John";
int age = 25;
String city = "Jakarta";

Aturan Penamaan Variabel

  • Harus dimulai dengan huruf, underscore (_), atau dollar sign ($)
  • Tidak boleh dimulai dengan angka
  • Case-sensitive (name ≠ Name)
  • Tidak boleh menggunakan reserved keywords
  • Gunakan nama yang deskriptif
// ✅ Valid
let userName = "John";
let _private = true;
let $price = 100;
let user2 = "Jane";

// ❌ Invalid
let 2user = "John";  // Dimulai dengan angka
let user-name = "John";  // Menggunakan dash
let class = "A";  // Reserved keyword

Naming Conventions

  • camelCase: userName, firstName (JavaScript, Java)
  • snake_case: user_name, first_name (Python)
  • PascalCase: UserName, FirstName (Classes)
  • UPPER_CASE: MAX_SIZE, API_KEY (Constants)

📦 Tipe Data

Tipe data menentukan jenis nilai yang bisa disimpan dalam variabel dan operasi apa yang bisa dilakukan pada nilai tersebut.

Tipe data dibagi menjadi 2 kategori:

  • Primitive (Primitif): Tipe data dasar
  • Complex (Kompleks): Tipe data yang terdiri dari multiple values

🔢 Tipe Data Primitif

1. Number (Angka)

Menyimpan nilai numerik (integer atau floating-point).

// JavaScript
let age = 25;           // Integer
let price = 99.99;      // Float
let negative = -10;     // Negative number
let scientific = 2e3;   // 2000 (scientific notation)

// Operasi matematika
let sum = 10 + 5;       // 15
let diff = 10 - 5;      // 5
let product = 10 * 5;   // 50
let quotient = 10 / 5;  // 2
let remainder = 10 % 3; // 1 (modulo)

2. String (Teks)

Menyimpan teks atau karakter.

// JavaScript
let name = "John Doe";
let greeting = 'Hello World';
let template = `Hello, ${name}!`; // Template literal

// String operations
let fullName = "John" + " " + "Doe"; // Concatenation
let length = name.length;            // 8
let upper = name.toUpperCase();      // "JOHN DOE"
let lower = name.toLowerCase();      // "john doe"
let char = name[0];                  // "J" (indexing)

3. Boolean

Menyimpan nilai true atau false.

let isActive = true;
let isLoggedIn = false;

// Comparison operators
let result1 = 5 > 3;    // true
let result2 = 5 < 3;    // false
let result3 = 5 === 5;  // true
let result4 = 5 !== 3;  // true

// Logical operators
let and = true && false;  // false
let or = true || false;   // true
let not = !true;          // false

4. Null

Merepresentasikan "tidak ada nilai" secara intentional.

let data = null;  // Explicitly no value
console.log(data); // null

5. Undefined

Variabel yang sudah dideklarasi tapi belum diberi nilai.

let x;
console.log(x); // undefined

let obj = {};
console.log(obj.name); // undefined (property doesn't exist)
💡 Null vs Undefined:
- null: Intentionally empty (programmer set it)
- undefined: Variable declared but not assigned (default)

🎨 Tipe Data Kompleks

1. Array

Menyimpan multiple values dalam satu variabel.

// Deklarasi array
let numbers = [1, 2, 3, 4, 5];
let fruits = ["apple", "banana", "orange"];
let mixed = [1, "text", true, null];

// Akses element
console.log(fruits[0]);  // "apple"
console.log(fruits[1]);  // "banana"

// Array methods
fruits.push("grape");     // Tambah di akhir
fruits.pop();             // Hapus dari akhir
fruits.unshift("mango");  // Tambah di awal
fruits.shift();           // Hapus dari awal

// Array properties
console.log(fruits.length); // Jumlah element

// Iterasi
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

fruits.forEach(fruit => {
    console.log(fruit);
});

2. Object

Menyimpan data dalam bentuk key-value pairs.

// Deklarasi object
let person = {
    name: "John Doe",
    age: 25,
    city: "Jakarta",
    isStudent: true
};

// Akses property
console.log(person.name);      // "John Doe" (dot notation)
console.log(person["age"]);    // 25 (bracket notation)

// Modify property
person.age = 26;
person.email = "john@example.com"; // Tambah property baru

// Delete property
delete person.city;

// Object methods
let user = {
    name: "John",
    greet: function() {
        return `Hello, ${this.name}!`;
    }
};

console.log(user.greet()); // "Hello, John!"

3. Function

Block of code yang bisa dipanggil berulang kali.

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Function expression
const add = function(a, b) {
    return a + b;
};

// Arrow function
const multiply = (a, b) => a * b;

// Usage
console.log(greet("John"));    // "Hello, John!"
console.log(add(5, 3));        // 8
console.log(multiply(4, 2));   // 8

🔄 Type Conversion

Implicit Conversion (Coercion)

// JavaScript automatically converts types
console.log("5" + 3);    // "53" (number to string)
console.log("5" - 3);    // 2 (string to number)
console.log("5" * "2");  // 10 (both to numbers)
console.log(true + 1);   // 2 (boolean to number)

Explicit Conversion

// String to Number
let str = "123";
let num1 = Number(str);      // 123
let num2 = parseInt(str);    // 123
let num3 = parseFloat("3.14"); // 3.14

// Number to String
let num = 123;
let str1 = String(num);      // "123"
let str2 = num.toString();   // "123"

// To Boolean
Boolean(1);        // true
Boolean(0);        // false
Boolean("");       // false
Boolean("text");   // true

🔍 Checking Types

// typeof operator
console.log(typeof 42);           // "number"
console.log(typeof "hello");      // "string"
console.log(typeof true);         // "boolean"
console.log(typeof undefined);    // "undefined"
console.log(typeof null);         // "object" (known bug)
console.log(typeof []);           // "object"
console.log(typeof {});           // "object"
console.log(typeof function(){}); // "function"

// Array check
Array.isArray([]);  // true
Array.isArray({});  // false

✨ Kesimpulan

Memahami tipe data dan variabel adalah fondasi penting dalam pemrograman. Setiap bahasa pemrograman memiliki tipe data yang mirip dengan sedikit perbedaan. Praktikkan penggunaan berbagai tipe data untuk memahami kapan menggunakan yang mana. Ingat, memilih tipe data yang tepat akan membuat program lebih efisien dan mudah dipahami!
