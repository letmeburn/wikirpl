🎯 Apa itu Semantic HTML?

Semantic HTML adalah penggunaan tag HTML yang memberikan makna (meaning) pada konten, bukan hanya tampilan. Tag semantic mendeskripsikan tujuan dan struktur konten dengan jelas.

✨ Manfaat Semantic HTML

1. Accessibility (Aksesibilitas)

Screen readers dapat memahami struktur halaman dengan lebih baik, membantu pengguna dengan disabilitas visual.

2. SEO (Search Engine Optimization)

Search engines lebih mudah memahami konten dan struktur website, meningkatkan ranking.

3. Maintainability

Kode lebih mudah dibaca dan dipahami oleh developer lain.

4. Consistency

Struktur yang konsisten di seluruh website.

💡 Best Practice:
Gunakan semantic elements untuk struktur, bukan styling. Styling dilakukan dengan CSS. Semantic HTML membuat kode lebih meaningful dan accessible.
