🎯 Pengenalan VS Code

Visual Studio Code (VS Code) adalah code editor gratis dan open-source dari Microsoft yang sangat populer di kalangan developer. VS Code ringan, cepat, dan memiliki ekosistem extension yang sangat kaya.

⌨️ Keyboard Shortcuts Penting

Navigasi & Pencarian

  • Ctrl + P - Quick Open (cari file dengan cepat)
  • Ctrl + Shift + F - Search di seluruh project
  • Ctrl + G - Go to Line
  • Ctrl + Tab - Switch antar file yang terbuka
  • Alt + ← / → - Navigate back/forward

Editing

  • Ctrl + D - Select next occurrence (multi-cursor)
  • Alt + Click - Add cursor di posisi tertentu
  • Ctrl + Shift + K - Delete line
  • Alt + ↑ / ↓ - Move line up/down
  • Shift + Alt + ↑ / ↓ - Copy line up/down
  • Ctrl + / - Toggle comment
// Contoh penggunaan multi-cursor (Ctrl + D)
const name = "John";
const age = 25;
const city = "Jakarta";

// Pilih "const" lalu tekan Ctrl + D beberapa kali
// untuk select semua occurrence dan edit sekaligus

View & Layout

  • Ctrl + B - Toggle sidebar
  • Ctrl + ` - Toggle terminal
  • Ctrl + \ - Split editor
  • Ctrl + 1/2/3 - Focus ke editor group

🔌 Extensions Wajib

1. Prettier - Code Formatter

Auto-format code agar konsisten dan rapi.

// settings.json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}

2. ESLint

Linting untuk JavaScript/TypeScript, mendeteksi error dan bad practices.

3. Live Server

Development server dengan auto-reload untuk HTML/CSS/JS.

4. GitLens

Supercharge Git capabilities, lihat history, blame, dan lainnya.

5. Auto Rename Tag

Otomatis rename pasangan HTML/XML tag.

6. Path Intellisense

Autocomplete untuk file paths.

7. Bracket Pair Colorizer 2

Warnai bracket berpasangan agar mudah dibaca (built-in di VS Code versi terbaru).

💡 Tips Produktivitas

1. Command Palette (Ctrl + Shift + P)

Akses semua command VS Code dengan cepat. Ketik apa yang ingin dilakukan.

2. Snippets

Buat custom snippets untuk code yang sering dipakai.

// File > Preferences > User Snippets
{
  "Console Log": {
    "prefix": "clg",
    "body": [
      "console.log('$1:', $1);"
    ],
    "description": "Console log with label"
  }
}

3. Emmet

Tulis HTML dengan cepat menggunakan abbreviations.


div.container>ul>li*3>a

4. Settings Sync

Sync settings, extensions, dan shortcuts ke cloud (built-in dengan GitHub account).

5. Zen Mode (Ctrl + K Z)

Mode fokus tanpa distraksi, sembunyikan semua UI kecuali editor.

💡 Pro Tip:
Pelajari shortcuts sedikit demi sedikit. Fokus pada yang paling sering dipakai dulu. Dalam beberapa minggu, produktivitas coding Anda akan meningkat drastis!

6. Integrated Terminal

Gunakan terminal langsung di VS Code, bisa split multiple terminals.

7. Workspace Settings

Buat settings khusus per project di .vscode/settings.json.

✨ Kesimpulan

VS Code adalah tool yang sangat powerful untuk developer. Dengan menguasai shortcuts, memanfaatkan extensions yang tepat, dan mengoptimalkan workflow, produktivitas coding Anda akan meningkat signifikan. Jangan ragu untuk eksplorasi fitur-fitur lainnya dan customize sesuai kebutuhan Anda!
