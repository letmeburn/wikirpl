🎯 Apa itu OOP?

Object Oriented Programming (OOP) adalah paradigma pemrograman yang mengorganisir kode berdasarkan objek yang merepresentasikan entitas di dunia nyata.

🏛️ 4 Pilar OOP

1. Encapsulation (Enkapsulasi)

Menyembunyikan detail implementasi dan hanya menampilkan interface yang diperlukan.

class BankAccount {
    #balance = 0; // Private property
    
    deposit(amount) {
        if (amount > 0) {
            this.#balance += amount;
        }
    }
    
    getBalance() {
        return this.#balance;
    }
}

2. Inheritance (Pewarisan)

Class child mewarisi properties dan methods dari class parent.

class Animal {
    constructor(name) {
        this.name = name;
    }
    
    speak() {
        console.log(`${this.name} makes a sound`);
    }
}

class Dog extends Animal {
    speak() {
        console.log(`${this.name} barks`);
    }
}

const dog = new Dog('Buddy');
dog.speak(); // Output: Buddy barks

3. Polymorphism (Polimorfisme)

Kemampuan objek untuk mengambil banyak bentuk. Method yang sama bisa memiliki implementasi berbeda.

4. Abstraction (Abstraksi)

Menyembunyikan kompleksitas dan hanya menampilkan fitur penting.

💡 Keuntungan OOP:
✅ Kode lebih terorganisir dan mudah dipelihara
✅ Reusability tinggi
✅ Lebih mudah di-debug
✅ Cocok untuk proyek besar

💻 Contoh Lengkap

// Contoh sistem perpustakaan
class Book {
    constructor(title, author, isbn) {
        this.title = title;
        this.author = author;
        this.isbn = isbn;
        this.isAvailable = true;
    }
    
    borrow() {
        if (this.isAvailable) {
            this.isAvailable = false;
            return true;
        }
        return false;
    }
    
    return() {
        this.isAvailable = true;
    }
}

class Library {
    constructor() {
        this.books = [];
    }
    
    addBook(book) {
        this.books.push(book);
    }
    
    findBook(isbn) {
        return this.books.find(book => book.isbn === isbn);
    }
}

// Penggunaan
const library = new Library();
const book1 = new Book('Clean Code', 'Robert Martin', '123');
library.addBook(book1);
book1.borrow();

Kesimpulan: OOP adalah paradigma powerful yang membuat kode lebih maintainable dan scalable. Kuasai 4 pilar OOP untuk menjadi developer yang lebih baik!
