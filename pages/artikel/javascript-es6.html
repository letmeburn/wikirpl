🎯 Pengenalan ES6

ECMAScript 6 (ES6) atau ES2015 adalah update besar JavaScript yang membawa banyak fitur modern. ES6 membuat kode JavaScript lebih clean, readable, dan powerful.

📦 Let & Const

ES6 memperkenalkan let dan const sebagai alternatif var dengan block scope.

// var: function scope
var x = 10;
if (true) {
    var x = 20;
}
console.log(x); // 20

// let: block scope
let y = 10;
if (true) {
    let y = 20;
    console.log(y); // 20
}
console.log(y); // 10

// const: tidak bisa reassign
const PI = 3.14;
// PI = 3.15; // Error!

// tapi object/array bisa dimutasi
const user = { name: "John" };
user.name = "Jane"; // OK
user.age = 25; // OK
💡 Best Practice:
Gunakan const by default. Gunakan let hanya jika nilai akan berubah. Hindari var.

🏹 Arrow Functions

Syntax lebih ringkas untuk menulis function dan memiliki lexical this.

// Traditional function
function add(a, b) {
    return a + b;
}

// Arrow function
const add = (a, b) => a + b;

// Dengan satu parameter, kurung bisa dihilangkan
const square = x => x * x;

// Dengan multiple statements
const greet = name => {
    const message = `Hello, ${name}!`;
    return message;
};

// Lexical this
const person = {
    name: "John",
    hobbies: ["coding", "gaming"],
    showHobbies() {
        this.hobbies.forEach(hobby => {
            console.log(`${this.name} likes ${hobby}`);
            // 'this' refers to person object
        });
    }
};

📤 Destructuring

Extract values dari array atau object dengan syntax yang clean.

Array Destructuring

// Traditional
const colors = ["red", "green", "blue"];
const first = colors[0];
const second = colors[1];

// Destructuring
const [first, second, third] = colors;

// Skip elements
const [, , third] = colors;

// Default values
const [a, b, c = "yellow"] = ["red", "green"];
console.log(c); // "yellow"

// Swap variables
let x = 1, y = 2;
[x, y] = [y, x];

Object Destructuring

const user = {
    name: "John",
    age: 25,
    email: "john@example.com"
};

// Traditional
const name = user.name;
const age = user.age;

// Destructuring
const { name, age, email } = user;

// Rename variables
const { name: userName, age: userAge } = user;

// Default values
const { name, country = "Indonesia" } = user;

// Nested destructuring
const person = {
    name: "John",
    address: {
        city: "Jakarta",
        country: "Indonesia"
    }
};
const { address: { city, country } } = person;

🌟 Spread & Rest Operator

Spread Operator (...)

Expand array atau object.

// Array
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = [...arr1, ...arr2]; // [1,2,3,4,5,6]

// Copy array
const original = [1, 2, 3];
const copy = [...original];

// Object
const user = { name: "John", age: 25 };
const updatedUser = { ...user, age: 26, city: "Jakarta" };

// Function arguments
const numbers = [1, 2, 3];
Math.max(...numbers); // 3

Rest Operator (...)

Collect multiple elements menjadi array.

// Function parameters
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}
sum(1, 2, 3, 4); // 10

// Array destructuring
const [first, second, ...rest] = [1, 2, 3, 4, 5];
console.log(rest); // [3, 4, 5]

// Object destructuring
const { name, ...otherProps } = { 
    name: "John", 
    age: 25, 
    city: "Jakarta" 
};
console.log(otherProps); // { age: 25, city: "Jakarta" }

📝 Template Literals

String dengan embedded expressions dan multi-line support.

// Traditional
const name = "John";
const greeting = "Hello, " + name + "!";

// Template literals
const greeting = `Hello, ${name}!`;

// Expressions
const a = 5, b = 10;
console.log(`Sum: ${a + b}`); // "Sum: 15"

// Multi-line strings
const html = `
    

        
${name}

        
Welcome to our site!

    

`;

// Tagged templates
function highlight(strings, ...values) {
    return strings.reduce((result, str, i) => {
        return `${result}${str}${values[i] || ''}`;
    }, '');
}
const user = "John";
const message = highlight`Hello ${user}, you have ${5} messages`;
// "Hello John, you have 5 messages"

🎁 Fitur ES6 Lainnya

Default Parameters

function greet(name = "Guest") {
    return `Hello, ${name}!`;
}
greet(); // "Hello, Guest!"
greet("John"); // "Hello, John!"

Enhanced Object Literals

const name = "John";
const age = 25;

// Shorthand property
const user = { name, age };

// Method shorthand
const obj = {
    greet() {
        return "Hello!";
    }
};

// Computed property names
const key = "email";
const user = {
    [key]: "john@example.com"
};

✨ Kesimpulan

ES6 membawa banyak fitur yang membuat JavaScript lebih modern dan developer-friendly. Fitur-fitur seperti arrow functions, destructuring, spread operator, dan template literals sudah menjadi standar dalam development JavaScript modern. Pelajari dan gunakan fitur-fitur ini untuk menulis kode yang lebih clean dan maintainable!
