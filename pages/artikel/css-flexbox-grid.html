Menguasai CSS Flexbox dan Grid Layout

🎯 Pengenalan

Flexbox dan Grid adalah dua sistem layout modern di CSS yang membuat pembuatan layout responsive menjadi jauh lebih mudah dibanding metode tradisional seperti float atau positioning.

📦 CSS Flexbox

Flexbox adalah layout model satu dimensi (baris atau kolom) yang sangat cocok untuk mendistribusikan space dan align items dalam container.

Konsep Dasar Flexbox

.container {
    display: flex; /* Aktifkan flexbox */
}

Flex Container Properties

flex-direction

Menentukan arah main axis.

.container {
    display: flex;
    flex-direction: row; /* default: horizontal */
    /* flex-direction: column; vertikal */
    /* flex-direction: row-reverse; */
    /* flex-direction: column-reverse; */
}

justify-content

Align items sepanjang main axis.

.container {
    display: flex;
    justify-content: flex-start; /* default */
    /* justify-content: center; */
    /* justify-content: flex-end; */
    /* justify-content: space-between; */
    /* justify-content: space-around; */
    /* justify-content: space-evenly; */
}

align-items

Align items sepanjang cross axis.

.container {
    display: flex;
    align-items: stretch; /* default */
    /* align-items: center; */
    /* align-items: flex-start; */
    /* align-items: flex-end; */
    /* align-items: baseline; */
}

flex-wrap

Mengatur apakah items boleh wrap ke baris baru.

.container {
    display: flex;
    flex-wrap: nowrap; /* default */
    /* flex-wrap: wrap; */
    /* flex-wrap: wrap-reverse; */
}

Flex Item Properties

flex-grow

Menentukan seberapa besar item akan grow relatif terhadap item lain.

.item {
    flex-grow: 0; /* default, tidak grow */
    flex-grow: 1; /* grow sama rata */
    flex-grow: 2; /* grow 2x lebih besar */
}

flex-shrink

Menentukan seberapa besar item akan shrink jika tidak cukup space.

flex-basis

Ukuran default item sebelum space didistribusikan.

Shorthand: flex

.item {
    flex: 1; /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */
    flex: 0 1 auto; /* default */
    flex: 2 1 200px; /* grow: 2, shrink: 1, basis: 200px */
}

Contoh Praktis Flexbox

/* Navbar dengan Flexbox */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
}

.nav-links {
    display: flex;
    gap: 1rem;
}

/* Card Layout */
.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

.card {
    flex: 1 1 300px; /* grow, shrink, min-width */
}

/* Centering */
.center-box {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

🎨 CSS Grid

CSS Grid adalah layout system dua dimensi (baris dan kolom) yang powerful untuk membuat complex layouts.

Konsep Dasar Grid

.container {
    display: grid;
    grid-template-columns: 200px 200px 200px; /* 3 kolom */
    grid-template-rows: 100px 100px; /* 2 baris */
    gap: 10px; /* spacing antar cells */
}

Grid Container Properties

grid-template-columns & rows

.container {
    display: grid;
    
    /* Fixed sizes */
    grid-template-columns: 200px 300px 200px;
    
    /* Fractional units (fr) */
    grid-template-columns: 1fr 2fr 1fr; /* ratio 1:2:1 */
    
    /* Repeat */
    grid-template-columns: repeat(3, 1fr); /* 3 kolom sama besar */
    
    /* Auto-fill & Auto-fit */
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    
    /* Mix */
    grid-template-columns: 200px 1fr 2fr;
}

gap

.container {
    display: grid;
    gap: 20px; /* row dan column gap */
    /* atau */
    row-gap: 20px;
    column-gap: 10px;
}

justify-items & align-items

.container {
    display: grid;
    justify-items: center; /* horizontal alignment */
    align-items: center; /* vertical alignment */
}

Grid Item Properties

grid-column & grid-row

Menentukan posisi dan span item.

.item {
    /* Span 2 kolom */
    grid-column: span 2;
    
    /* Dari kolom 1 ke 3 */
    grid-column: 1 / 3;
    
    /* Shorthand */
    grid-column: 1 / span 2;
    
    /* Rows */
    grid-row: 1 / 3;
    grid-row: span 2;
}

grid-area

Shorthand untuk row-start / column-start / row-end / column-end.

.item {
    grid-area: 1 / 1 / 3 / 3;
}

Contoh Praktis Grid

/* Responsive Card Grid */
.card-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 2rem;
}

/* Dashboard Layout */
.dashboard {
    display: grid;
    grid-template-columns: 250px 1fr;
    grid-template-rows: 60px 1fr 50px;
    grid-template-areas:
        "sidebar header"
        "sidebar main"
        "sidebar footer";
    height: 100vh;
}

.sidebar { grid-area: sidebar; }
.header { grid-area: header; }
.main { grid-area: main; }
.footer { grid-area: footer; }

/* Holy Grail Layout */
.holy-grail {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
}

.header { grid-column: 1 / -1; }
.footer { grid-column: 1 / -1; }

⚖️ Kapan Menggunakan Apa?

Gunakan Flexbox untuk:

  • Layout satu dimensi (baris atau kolom)
  • Navigation bars
  • Card layouts sederhana
  • Centering elements
  • Distribusi space yang fleksibel

Gunakan Grid untuk:

  • Layout dua dimensi (baris dan kolom sekaligus)
  • Complex page layouts
  • Dashboard layouts
  • Gallery/masonry layouts
  • Ketika butuh precise control atas rows dan columns
💡 Pro Tip:
Flexbox dan Grid bukan kompetitor, tapi komplemen! Sering kali kita menggunakan Grid untuk overall page layout, dan Flexbox untuk components di dalamnya.

✨ Kesimpulan

Flexbox dan Grid adalah tools essential untuk modern web development. Flexbox cocok untuk layout satu dimensi dan distribusi space yang fleksibel, sementara Grid perfect untuk complex two-dimensional layouts. Kuasai keduanya untuk membuat layouts yang responsive dan maintainable!
